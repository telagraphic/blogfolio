Modern Security Guide

6/2018

Modern Security for Everyday Users

I spent a month researching all things security related, spent another month building a website guide for average non-techie users like your parents, friends and anyone clueless on where to get started.

It's almost at version 1, it's a project still in progress with further details and refinements coming before end of year. Many UI and UX changes are pending from feedback. Borrowed navigation concept from Uber.

Security is Hard, Teaching It is Tougher

Half way through making the site, I realized the overall experience wasn't really working. I had the hierachy figured out, all the different tech layers mapped out but the guide was missing the human element. People have questions about security. A Q&A style guide that answered specific questions and pain points would be a better approach.

I realized that my guide was half my personal cliff notes attempting to be an in-depth security assistant filled with endless attack vectors from all types of hackers. Average people don't have the time to secure their digitial lives.

Enter Pi-Hole

Network level security is where you get the most bang for your buck. A couple hours and about $80 can get you a network level DNS sinkhole right on a Raspberry Pi. Pi-Hole is an open source network wide ad blocker that sits between your router and home network. You can also set up VPN, DNS Server and DNS over HTTPS services right on the Pi.
