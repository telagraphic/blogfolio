Code Clothes

2021

Pitch

Computer Code + Streetwear Clothing

The Idea

Copy and paste computer code and related themes onto everyday clothing. Design patterns and color palettes make technical geek look interesting and visually appealing.

Domain Names

  • codeclothes.com

Alternate Names

  • Copy, Paste, Wear
  • Save to a shirt
  • Print and Wear
  • console.log(shirt)

Starting Ideas

  • UI Interface Shirt - grid of layout ui components
  • Git branches - pattern of git branches
  • SQL Shirt - sql statements arranged in blocks overlayed with database schema's
  • CSS - nested css rules that create triangular generative art patterns
  • Impossible HTML - nested html elements that follow a generative art pattern
  • Javascript - compressed code

Concepts

Take any computer language or concept and find a way to make a cool pattern or theme that can be printed onto fabric material and then sewn into a wearable item.

Designs will be "all over print" not one graphic artwork on the chest. Could offer custom sizing for an even better fit.

Emphasis on the opposite idea of fast fashion, clothing with a long shelf life.
