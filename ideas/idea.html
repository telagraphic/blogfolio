Budget Card

6/2018

Pitch

FIRE Community + Prepaid Card + Anti Credit Card = Budget Card

Replace your cards' credit limit with a budget limit that turns the card off once your weekly budget is met. Faced with the very real 'threat' of a non-working credit card, you are forced to change your spending habits while still getting credit for your purchases.

Problem Statement + Unique Value Proposition = Pitch

The Benefit is not what you are selling, it what your selling means to the people you are selling to. People buy this because...

The Differentiator is what is different between you and your competitors.

We solve [problem] by providing [advantage], to help [target] accomplish [target’s goal]. We make money by charging [customers] to get [benefit].

Concept

Flexbox's spacing algorithm isn't intuitive when you are first learning, but several CSS experts have written great articles explaining the math behind how growing and shrinking work.

It inspired me to build a tool I call the Flexulator for seeing how this formula works on flex items. Flexulator is a tool in progress, but I have completed most of the javascript code that displays the underlying flex math.

I have plans to fix up the UI, fix performance problems like grouping read/writes to prevent layout thrashing, and refactor the 400 line javascript file.

Business Model Canvas

In short, The Uncertainty Machine is a web-based tool to measure uncertainty for many different applications , some being for the age of a meteorite and for electrical resistance measurements. I’m definitely no scientist and really have no idea how to use this tool. I really wanted to make it look better and enhance the overall user experience. This wan’t a simple charity design case, I wanted to learn some skills like mobile first design, UX/UI concepts and applications, use grid and flexbox, and learn more about forms.

Here is the original version that I based my redesign on. I spent a couple of days auditing the UX/UI and made a plan of action that cut down the number of ‘steps’ from 7 to grouping them into 3 main actions, fixed up the form UI and added field validations and made it responsive for mobile with CSS Grid. The NIST scientists implemented my UX changes on the new version 1.4 release! Some resources I found exceptionally helpful are UXMovement for insightful UX best practices, WTFForms for a modern approach to styling form inputs, and CSS Tricks Grid Guide.
